# Restaurant Management Desktop Application

A desktop-based Restaurant Management System built using Electron.js, HTML, CSS, JavaScript, and Supabase. The application helps restaurant owners and staff efficiently manage tables, customer orders, order history, and sales records through an intuitive user interface.

## Features

* User Authentication
* Table Management
* Order Creation and Tracking
* Order History Management
* Food Managment with Category
* Edit and Delete Orders
* Real-time Data Storage with Supabase
* Sales Monitoring Dashboard
* Staff Managment
* Desktop Application Support (Windows EXE)

## Tech Stack

* Electron.js
* JavaScript (ES6+)
* HTML5
* CSS3
* Supabase

## Key Highlights

* Cloud-based database integration using Supabase
* Real-time CRUD operations
* Desktop application packaging with Electron Builder
* User-friendly interface for restaurant operations
* Multi-user support through a centralized database

## Purpose

This project was developed to gain practical experience in desktop application development, database integration, and real-world business management solutions. It demonstrates the implementation of authentication, data management, and cloud-based synchronization in a production-style application.
